# contest-winner

### Install
```
$ git clone https://github.com/shavah/contest-winner.git
$ cd contest-winner
$ lein test
$ lein run
```

### Dependencies
* [clj-twitter-api](https://github.com/adamwynne/twitter-api)
* [clj-http](https://github.com/dakrone/clj-http)
